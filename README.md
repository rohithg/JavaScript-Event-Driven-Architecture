# JavaScript Event-Driven Architecture

Advanced event system with middleware, priorities, and async support.

## Features

- Event emitter pattern
- Pub-sub architecture
- Middleware support
- Priority-based execution
- Event history
- Async/await support

## JavaScript Concepts

- Closures and scope
- Promises and async/await
- Event loop
- Prototype chain
- Symbol for unique IDs
- Map and Set

## Running

```bash
node event-system.js
```

## Use Cases

- Microservices communication
- UI event handling
- Plugin systems
- Workflow orchestration
